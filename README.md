# Connect four game
[![Travis](https://travis-ci.org/mareksabo/game-connect-four.svg)](https://travis-ci.org/mareksabo/game-connect-four)

School project at HTWG school in a _Software architecture class.

### Project description

Project represents [connect four](https://en.wikipedia.org/wiki/Connect_Four) game
with eligible dimensions. Game could be played in either graphical or text mode and on localhost:8080 if the Akka-Http-view is used.
